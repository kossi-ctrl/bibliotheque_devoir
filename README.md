# BibliothèqueX

Un projet de gestion de contenus pour une bibliothèque, incluant des pages de livres, événements et contacts.


# Installation

1. Cloner le dépôt :

`git clone https://github.com/kossi-ctrl/bibliotheque_devoir.git`

2. Ouvrir le projet dans Visual Studio Code (ou tout autre éditeur).  
3. Ouvrir les fichiers HTML dans un navigateur pour tester le site.

# Fonctionnalités

- Page d’accueil avec présentation de la bibliothèque.  
- Gestion et affichage des livres via `livres.html` et `livres.json`.  
- Gestion des événements avec `evenements.html` et `evenements.json`.  
- Page de contact avec formulaire basique.  
- Tous les styles gérés via CSS et les données dynamiques via JSON (JavaScript si applicable).

# Technologies utilisées

- HTML5  
- CSS3  
- JavaScript  
- JSON (pour les données de livres et événements)
